# HU-210 — API CRUD Liquidaciones
**Sprint:** 2 | **Rama:** B | **Alta**

## POST crea liquidación
1. Valida contrato pertenece al usuario
2. Unique (contrato_id + periodo)
3. Lee indicadores del mes
4. Lee contrato (sueldo, AFP, horas)
5. Calcula motor v3.1
6. Guarda calculo JSONB + campos desnormalizados
7. Auto-rellena RIMA desde liquidación anterior
