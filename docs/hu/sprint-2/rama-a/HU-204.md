# HU-204 — Migración Stripe → Transbank (Infraestructura)
**Sprint:** 2 | **Rama:** A | **Alta** | **CAMBIO DE DISEÑO**

## Contexto
Se reemplaza Stripe por Transbank Webpay Plus como medio de pago.
Transbank NO tiene suscripciones nativas ni billing portal.
El cobro es por transacción individual + renovación manual del usuario.

**LEER ANTES:** `docs/MIGRACION_STRIPE_A_TRANSBANK.md`

## Tareas

### 1. Limpiar Stripe
- Eliminar `src/lib/stripe/client.ts`
- Eliminar `src/app/api/webhooks/stripe/route.ts`
- Eliminar `src/app/api/billing/portal/route.ts`
- Eliminar dependencias: `npm uninstall stripe @stripe/stripe-js`

### 2. Instalar Transbank
- `npm install transbank-sdk`
- Crear `src/lib/transbank/client.ts`:
  ```typescript
  import { WebpayPlus, Environment } from 'transbank-sdk'

  const environment = process.env.TBK_ENVIRONMENT === 'production'
    ? Environment.Production
    : Environment.Integration

  export const webpayTransaction = new WebpayPlus.Transaction(
    new WebpayPlus.Options(
      process.env.TBK_COMMERCE_CODE!,
      process.env.TBK_API_KEY!,
      environment
    )
  )

  export const PLAN_PRICES: Record<string, number> = {
    Pro_Mensual: 9900,
    Pro_Anual: 95040,
  }

  export const PLAN_DURATIONS: Record<string, number> = {
    Pro_Mensual: 30,
    Pro_Anual: 365,
  }
  ```

### 3. Actualizar ENV
En `.env.local` agregar:
```
TBK_COMMERCE_CODE=597055555532
TBK_API_KEY=579B532A7440BB0C9079DED94D31EA1615BACEB56610332264630D42D0A36B1C
TBK_ENVIRONMENT=integration
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
Eliminar las variables `STRIPE_*`.

### 4. Migración DB
Crear `supabase/migrations/002_stripe_to_transbank.sql`:
```sql
ALTER TABLE user_billing RENAME COLUMN stripe_customer_id TO tbk_order_id;
ALTER TABLE user_billing RENAME COLUMN stripe_subscription_id TO tbk_token;

COMMENT ON COLUMN user_billing.tbk_order_id IS 'Último buy_order de Transbank';
COMMENT ON COLUMN user_billing.tbk_token IS 'Token de última transacción Webpay exitosa';
```
Ejecutar con MCP Supabase o directamente en dashboard.
