# HU-201 — Generar Contrato Completo E2E
**Sprint:** 2 | **Rama:** A | **Alta**

## Flujo
1. Leer contrato de BD
2. Determinar condicionales (tipo/jornada/doc)
3. renderTemplate(plantilla, variables)
4. generatePDF(html)
5. uploadPDF → Supabase Storage
6. enviarEmailContrato → Resend
7. Update contrato: estado="enviado", pdf_url=URL

## Archivos
- `src/app/api/contratos/[id]/generar-pdf/route.ts`
- `src/lib/pdf/generar-contrato.ts`

## Criterios de aceptación
- [ ] POST ejecuta flujo completo
- [ ] PDF descargable y profesional
- [ ] Email llega al empleador
- [ ] Estado cambia a "enviado", pdf_url guardado
