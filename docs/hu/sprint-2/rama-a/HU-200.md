# HU-200 — API CRUD Contratos
**Sprint:** 2 | **Rama:** A | **Alta**

## Archivos
- `src/app/api/contratos/route.ts` → GET (list), POST (create)
- `src/app/api/contratos/[id]/route.ts` → GET (detail)

## POST crea contrato
- estado: "borrador", genera token UUID
- Calcula total_horas_semanales y dias_semana desde horarios
- Si tipo_sueldo="liquido" → calcula sueldo_base con bisección

## Criterios de aceptación
- [ ] GET retorna solo contratos del usuario (RLS)
- [ ] POST crea con token + estado borrador
- [ ] GET /[id] retorna 404 si no pertenece
- [ ] Validación campos requeridos → 400
- [ ] Calcula horas y sueldo base automáticamente
