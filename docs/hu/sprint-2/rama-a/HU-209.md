## Tests
Crear `tests/billing/transbank.test.ts`:

### Casos de prueba
1. **Checkout crea transacción válida:**
   - Mock `webpayTransaction.create()` → retorna url + token
   - Verifica que billing_transactions se crea con status='pending'

2. **Callback aprueba transacción:**
   - Mock `webpayTransaction.commit()` → responseCode 0
   - Verifica user_billing actualizado con plan activo
   - Verifica billing_transactions.status = 'approved'

3. **Callback rechaza transacción:**
   - Mock `webpayTransaction.commit()` → responseCode !== 0
   - Verifica billing_transactions.status = 'rejected'
   - Verifica user_billing NO cambia

4. **Callback sin token_ws:**
   - Verifica redirect a /facturacion?status=cancelled

5. **checkPlanAccess con plan activo:**
   - Verifica features Pro habilitadas

6. **checkPlanAccess con plan expirado:**
   - Verifica { allowed: false }

7. **Cron expire-plans:**
   - Insertar user_billing con end_date pasada
   - Ejecutar cron
   - Verificar plan_status = 'expired'
