# HU-220 — Formulario Contrato Completo
**Sprint:** 2 | **Rama:** C | **Alta**

## Definición
/contratos/nuevo — 4 secciones con MultiStepForm:
1. Empleador (6 campos)
2. Trabajador (12+ con bifurcación RUT/Pasaporte + es_pensionado, cargas)
3. Jornada (16 campos horarios)
4. Condiciones (11 con bifurcación tipo sueldo)

Al completar → POST /api/contratos → POST generar-pdf
