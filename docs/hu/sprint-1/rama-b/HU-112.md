# HU-112 — API Calcular Liquidación
**Sprint:** 1 | **Rama:** B | **Alta**

## Definición
POST /api/calcular-liquidacion — público (sin auth)
- Lee indicadores del mes desde indicadores_previsionales
- Guarda en tabla simulaciones si incluye email
- Retorna ResultadoLiquidacion

## Request
```json
{ "sueldo_pactado": 500000, "tipo_sueldo": "liquido", "afp": "Uno", "horas_semanales": 45,
  "colacion?": 0, "movilizacion?": 0, "cargas_familiares?": 0, "email?": "" }
```

## Response
ResultadoLiquidacion completo (200) o { error: string } (400/500)

## Archivos
`src/app/api/calcular-liquidacion/route.ts`

## Criterios de aceptación
- [ ] curl con datos válidos → sueldo_liquido correcto
- [ ] Campos faltantes → 400
- [ ] AFP inválida → 400
- [ ] Con email → guarda en simulaciones
- [ ] Indicadores de BD, no hardcodeados

## Dependencias
HU-110, HU-001
