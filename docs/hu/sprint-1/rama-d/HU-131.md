# HU-131 — Simulador Liquidación Público
**Sprint:** 1 | **Rama:** D | **Alta**

## Definición
/simulador — formulario estándar (no multi-step):
- Sueldo, tipo (bruto/líquido), AFP, horas, email (requeridos)
- Expandibles: colación, movilización, bono, cargas
- Submit → POST /api/calcular-liquidacion → resultado 3 secciones

## Criterios de aceptación
- [ ] Sin auth requerido
- [ ] Resultado: Sección 1 (sueldos), Sección 2 (cotizaciones), Sección 3 (costo empleador)
- [ ] Montos formateados CLP

- [ ] Responsive
