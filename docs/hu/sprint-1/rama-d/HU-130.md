# HU-130 — MultiStepForm Engine
**Sprint:** 1 | **Rama:** D | **Alta**

## Historia
Como usuario quiero un formulario paso a paso estilo Typeform.

## Definición
- `<MultiStepForm steps={FormStep[]} onComplete={(data) => void} />`
- 1 pregunta por pantalla, transición vertical (Framer Motion)
- Barra progreso superior verde #6fc8a0
- Enter avanza, botón Atrás retrocede
- Validación Zod por campo
- Bifurcaciones: step.showIf(answers)
- Guardado parcial localStorage
- **RESTRICCIÓN:** FormStep[] debe ser array plano serializable (futuro WhatsApp bot)

## Tipos
```typescript
interface FormStep {
  id: string; section: string; question: string; description?: string;
  type: 'text'|'email'|'number'|'date'|'select'|'multiselect'|'radio'|'textarea'|'rut';
  options?: {label: string; value: string}[];
  required: boolean; validation?: ZodSchema;
  showIf?: (answers: Record<string, any>) => boolean;
}
```

## Criterios de aceptación
- [ ] Renderiza primer paso al montar
- [ ] Enter/botón avanza (respeta showIf)
- [ ] Atrás retrocede
- [ ] Progreso proporcional a pasos visibles
- [ ] Campo requerido vacío → no avanza, error visible
- [ ] Tipo 'rut' valida y formatea
- [ ] Último paso → onComplete(allAnswers)
- [ ] Responsive
- [ ] Animación slide suave
